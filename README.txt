== Description ==

Clipboard Express is a multi-user clipboard interface plugin for WordPress.

Each user account has their own private clipboard that can be edited from a **Clipboard Express** section or floating module below the post.


== Installation ==

Installing and customization is easy and takes fewer than five minutes.

1. Upload the `clipboard-express.php` file to the `/wp-content/plugins/` directory
2. Activate the plugin *Clipboard Express* through the 'Plugins' menu in WordPress
3. You can access and edit your clipboard through `Write > Clipboard Express`, `Manage > Clipboard Express` or below the post module in either the posts and pages swrite and manage screens.

== Frequently Asked Questions ==

= Can I access other people's clipboards? =

No, you can only acces or edit your own clipboard. This goes for all user levels.

= Is there a public clipboard that I can use so everyone has access to? =

No, it may be built in as an option in the future but that is a ways down the road.
